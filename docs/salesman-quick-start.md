Salesperson Quick Start

- Open the app and sign in with your credentials.
- You will land on the large "Record Sale" quick action (Sales role).
- To sell:
  - Use the search box at the top to find a product by name or code.
  - Tap a product tile to select it.
  - Use the preset buttons (1, 2, 5, 10) or the + / - controls to set quantity.
  - (Optional) Enter a discount in ₹ if allowed.
  - Tap the big green "Sell" button to record the sale.
  - A confirmation message will appear with the sale total.
- Need to look up a product by code? Tap "Search" from the quick actions.
- Incoming shipments show under "Shipments" (tap to confirm receipt).

Tips for fast selling:
- Use the search bar (it auto-focuses) and tap the first matching tile.
- Use the preset quantity buttons for quick entry.
- The UI is simplified for touch devices and new users — if you need the advanced admin options, ask an admin.
